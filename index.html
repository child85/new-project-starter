AI Backend Setup

The browser page should never hold an AI API key. Point this page at an API Gateway endpoint backed by Lambda, then store the endpoint URL here for testing.

1 Create API Gateway trigger

In the Lambda function, add an API Gateway HTTP API trigger with a POST route named /enrich.

2 Keep secrets in Lambda

Add ANTHROPIC_API_KEY, ANTHROPIC_MODEL, and ALLOWED_ORIGIN in Lambda environment variables.

3 Paste endpoint here

Use the API Gateway invoke URL ending in /enrich. The site will call AWS first, then fall back to demo mode.

Next AWS step: Lambda -> Add trigger -> API Gateway -> HTTP API -> POST /enrich.
